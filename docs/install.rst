Installation

Installing |mobiusmc|
========================

The easiest way to install |mobiusmc| is with PyPi.

.. code-block:: bash

    pip install mobiusmc

To get the very latest version of the software, install it directly from GitHub.
    
.. code-block:: bash

    pip install git+https://github.com/dnidever/mobiusmc

Dependencies
============

- numpy
- scipy
- astropy
- matplotlib
- dlnpyutils
- corner
- emcee
- dill
